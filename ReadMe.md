Weather Forecast Application

Overview :

This project is a Weather Forecast Web Application built using HTML, Tailwind CSS, and JavaScript. It allows users to search for weather conditions of any city and also fetch weather data based on their current location.

The application displays real-time weather information along with a 5-day forecast in a clean and responsive UI.

Features :

1. Search weather by city name
2. Get weather using current location
3. Display temperature, humidity, wind speed, pressure, and feels-like temperature
4. 5 days forecast
5. Recently searched cities (Local storage)
6. Dynamic background based on weather conditions
7. Proper error handling with UI messages

Output :

The application shows:

1. Current weather details
2. Dynamic UI changes based on weather
3. 5-day forecast cards
